# Trems AI Email Templates

## Available Templates

### 1. `trems-services-template.html`
**Full-featured template** with detailed service descriptions, stats, and visual sections.
- Best for: Comprehensive service overviews
- Use when: Introducing all services at once

### 2. `trems-simple-template.html`
**Simplified template** optimized for promotional folders.
- Best for: Regular newsletters, follow-ups
- Use when: Quick updates, less visual clutter needed

## How to Use with Unlayer Editor

1. **Copy HTML**: Open the template file and copy the HTML content
2. **Import to Unlayer**: 
   - In your campaign editor, click "Import HTML"
   - Paste the HTML code
   - Unlayer will convert it to editable blocks
3. **Customize**: 
   - Edit text, colors, and layout
   - Add/remove sections
   - Update merge tags ({{firstName}}, etc.)
4. **Save**: Save the design for reuse

## Merge Tags Available

- `{{firstName}}` - Contact's first name
- `{{lastName}}` - Contact's last name
- `{{email}}` - Contact's email
- `{{fullName}}` - Full name
- `{{unsubscribe_url}}` - Auto-generated unsubscribe link
- Custom fields: `{{customFieldName}}` - Any custom field from contacts

## Customization Tips

1. **Colors**: Update `#2563eb` (blue) to your brand color
2. **Services**: Modify service descriptions to match your offerings
3. **Stats**: Update with your real metrics
4. **Links**: Replace `https://trems.hr/kontakt` with your actual URLs
5. **Contact**: Update email and phone in footer

## Best Practices

- Keep subject lines under 50 characters
- Test on mobile devices before sending
- Use merge tags for personalization
- Include clear, single CTA
- Keep content scannable (short paragraphs)
- Use bullet points for features

## Plain Text Fallback

For better deliverability, consider adding a plain text version. The system can send both HTML and plain text versions automatically.
